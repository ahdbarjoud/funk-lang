### Funk Variables
The type of each variable must be know at compile time.
```
// A semi is not important.
String wee = "Wooo";

Integer waa = 50
```

### Funk conditionals
```
if (a == 1) {
  // Do something
} else if (a == 2) {
  // Do something else
} else {
  // Do something else else
}
```
